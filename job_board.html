Job Board

This project is a Job Board application. An employeer can post a new job position. A job seeker can apply to any jobs posted. This is a full-stack Javascript project that uses PostgresSQL as the database. The application is deployed to Heroku. Files such as resumes uploaded are store on Cloudinary.
