ReactCommerce

This project is an Ecommerce web application built in React and Node.js. In this application, a user can search for a product, add that product to an online shopping cart, and then purchase all items in the cart through the Stripe API checkout system. Rather than retrieving data from a public API, the product data is stored and updated in the Context API. User data is stored on Atlas which is a cloud database. This project is deployed on to Heroku.
